用Java生成随机数

Posted on 2020-02-21 In 学习笔记 Views:

使用nextInt()方法

模拟需求:
班级需要一个点名器，输入班级最大学号，获得十个不同的随机学号
获取10个1-20之间的随机数，要求不能重复

其中一个运行结果：[10, 17, 1, 12, 18, 2, 15, 6, 4, 5]

总结：Random.nextInt()方法可以生成一个随机数，范围是2^-31 ~ 2^31 - 1，但在方法内传入参数(如m)后，随机数生成的范围变为[0,m)。

让用户输入范围[strat,end]
随机生成该范围内的一个随机数
